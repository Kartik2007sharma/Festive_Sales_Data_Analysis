# Diwali Sales Data Analysis

This project focuses on analyzing sales data during the Diwali festive season using Python. The goal is to extract meaningful business insights by studying customer behavior, product performance, and sales trends.

The complete analysis is provided in a Jupyter Notebook along with visualizations.

## Overview

Diwali is one of the biggest shopping seasons in India. This project analyzes festive sales data to understand purchasing patterns, identify high-performing products, and observe sales growth across different years and categories.

## Objective

The main objectives of this project are:
- To analyze sales trends during the Diwali season
- To understand customer purchasing behavior
- To identify top-selling products and categories
- To derive insights that can help in business decision-making

## Dataset

The dataset used in this project contains:
- Order and invoice details
- Product categories
- Sales amount and revenue
- Year and month information

Ensure the dataset is available in the correct directory before running the notebook.

## Tools and Technologies

The following tools and libraries are used in this project:

- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Analysis Performed

The project includes the following steps:
- Data cleaning and preprocessing
- Handling missing and duplicate values
- Exploratory data analysis
- Grouping and aggregation of sales data
- Data visualization to identify trends and patterns

## Key Insights

Some of the insights derived from the analysis include:
- Identification of high-revenue products
- Sales trends across different years
- Product category performance during Diwali
- Seasonal impact on customer purchases
